# Paper-Template

A template for writing paper drafts. Tested on Overleaf. 

## Structure

```headers/pkgs.tex```
Package includes go here.

```headers/defs.tex```
Shortcuts go here.

```headers/setup.tex```
Other setup commands go here.

```fig```
All figures go here.

```ref/ref.bib```
One stop shop for all citations.

```src/*.tex```
Individual source files for different sections.

```main.tex```
Main document compiled.

```main.bib```
References go here.

## How to compile

```pdflatex main.tex```
or use Overleaf.
